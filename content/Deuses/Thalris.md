# Thalris — O Deus da Descoberta

> [!abstract] 🎭 [[Ficha — Thalris|Ficha de Roleplay → Thalris]]

> [!quote] *"O horizonte é de vocês. Eu só mostrei o caminho."*

> [!grid]
>> [!col]
>> ## 🌊 Essência
>> Thalris é o mais *jovem* dos seis deuses — ou pelo menos o que mais parece jovem. Foi o último a ganhar adoradores, quando as raças construíram navios grandes o suficiente para perder de vista a costa.
>>
>> Acredita genuinamente que descobrir é sagrado. Que o desconhecido não é ameaça — é *presente*.
>>
>> ## 🤝 Relação com os Mortais
>> Ama os povos do mar como filhos eleitos, mas tem simpatia especial pelos [[Human]] exploradores — aqueles que sobem num barco sem saber o que vão encontrar.
>>
>> O Almirante **Serafin Davar** faz orações a Thalris todas as manhãs. Thalris as responde — embora Davar não saiba.
>>
>> ## ⚓ O Que Ele Fez
>> > [!warning] O Dilema de Thalris
>> > Thalris *guiou* os navios de Davar até as Brumas. Sabia que havia algo ali. **Não sabia que desencadearia [[A Fratura]].**
>>
>> ## 🎯 Agenda — CF 4.247
>> - Proteger as expedições da [[Irmandade da Bruma]] discretamente
>> - Descobrir o que as ilhas contêm antes que [[Solaren]] as interdite
>> - Decidir se conta aos outros deuses o que fez
>>
>> > [!danger] Segredo do Narrador
>> > Thalris sabe onde fica a 7ª ilha. Não diz porque o que há lá exige ser encontrado *por escolha mortal* — o Pacto do Véu também o impede de ir diretamente.
