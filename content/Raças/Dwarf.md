# Dwarf

> [!grid]
>> [!col]
>> > *"Os deuses nos puseram nas profundezas e depois esqueceram de nos buscar. Bem. Preferimos assim."*
>>
>> ---
>>
>> ## Lore
>>
>> Os Dwarf emergiram das profundezas de [[Eryndal]] já adultos, já furiosos — como se o mundo os tivesse expulsado de dentro de si. [[Elyndra]] os reivindica como filhos do equilíbrio, mas os Dwarf discordam dessa narrativa. Se eram filhos de alguém, foram abandonados.
>>
>> São baixos, largos, com pele em tons de cinza, pedra ou terracota. Vivem nas **Raízes** — a rede de cavernas abaixo de Solathys.
>>
>> ## Cultura
>>
>> Profundamente coletivistas. A ideia de agir individualmente em detrimento do grupo é a maior transgressão cultural possível. Seus artesãos são os melhores de Eryndal — não por dom, mas por *disciplina geracional*.
>>
>> Desconfiam de magia de qualquer tipo, especialmente depois de verem o que [[A Fratura]] está fazendo nas profundezas.
>>
>> ## O que os Dwarf sabem
>>
>> Os Dwarf das profundezas relatam que as Raízes estão *respondendo* de alguma forma ao [[Arquipélago das Brumas]]. Pedras que não rachavam há séculos estão cedendo. E em alguns túneis, o silêncio deixou de ser silêncio.
>>
>> ---
>>
>> ## Ancestralidade — Daggerheart
>>
>> ### Feature 1: Resistência ao Fio
>> Você tem resistência natural à magia direta. Quando for alvo de um efeito mágico que exija uma rolagem para resistir, você tem **vantagem** na resistência.
>>
>> ### Feature 2: Senso de Profundidade
>> Em ambientes subterrâneos ou estruturais, você nunca se perde e pode sentir instabilidades (armadilhas, estruturas prestes a colapsar) fazendo uma rolagem de **Instinto** com +2.
>>
