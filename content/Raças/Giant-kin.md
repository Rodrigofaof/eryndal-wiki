# Giant-kin

> [!grid]
>> [!col]
>> > *"Antes da Forja Primordial, o Vazio não estava vazio. Os Giant-kin são o que sobrou quando os deuses terminaram."*
>>
>> ---
>>
>> ## Lore
>>
>> Antes de [[Eryndal]] existir, o [[Cosmologia|Vazio]] tinha habitantes — formas massivas e instáveis que existiam sem propósito definido, como pensamentos sem pensador. Quando os seis deuses começaram a Forja Primordial, moldaram o Vazio em mundo. A maioria dessas formas foi dissolvida no processo.
>>
>> Algumas não foram.
>>
>> Os Giant-kin são os descendentes dessas formas — seres que carregam no sangue fragmentos do Vazio pré-criação. São altos (entre 2,5 e 3,5 metros), de constituição densa, com traços que lembram pedra ou metal natural, e uma presença física que faz o ar ao redor deles parecer mais pesado.
>>
>> ## Uma Raça Sem Deus
>>
>> Nenhum dos seis deuses reivindica os Giant-kin — porque nenhum os criou. Eles existem apesar dos deuses, não por causa deles. Isso cria uma liberdade incomum: Giant-kin não têm obrigações teológicas, não têm templos, não têm pecado definido por divindade.
>>
>> Os [[Dwarf]] os respeitam por isso. A [[Igreja de Solaren]] os tolera com desconforto evidente.
>>
>> ## Postura Atual
>>
>> Com [[A Fratura]] abrindo o [[Cosmologia|Véu]], os Giant-kin estão sentindo algo que nunca sentiram — um *chamado* vindo de algum lugar abaixo do mundo. Alguns descrevem como ouvir sua própria origem chamar de volta. É perturbador na exata medida em que deveria ser.
>>
>> ---
>>
>> ## Ancestralidade — Daggerheart
>> *Equivalente: Giant-kin*
>>
>> ### Feature 1: Porte do Vazio
>> Você conta como duas categorias de tamanho maior para propósitos de espaço e grapple. Seu dano desarmado é **1d10**. Criaturas de tamanho Médio ou menor têm desvantagem para resistir a seus ataques de empurrão ou derrubada.
>>
>> ### Feature 2: Memória Pré-Criação
>> *Uma vez por sessão*, você pode fazer uma pergunta sobre algo que existia *antes* de Eryndal. O Narrador deve responder com um fragmento de verdade — pode ser incompleto ou estranho, mas é genuíno.
>>
