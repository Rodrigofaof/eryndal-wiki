# Human

> [!grid]
>> [!col]
>> > *"A vantagem dos Human não é que vivem muito. É que nunca aprendem a ter medo."*
>> > — provérbio Elf, geralmente dito como insulto
>>
>> ---
>>
>> ## Lore
>>
>> Os Human são a raça dominante de [[Eryndal]] — e ninguém sabe por quê eles existem. Nenhum deus reivindica tê-los criado. Os [[Elf]] acreditam que são um acidente. Os [[Dwarf]] acreditam que foram criados por [[Azúrael]] antes do aprisionamento. Os próprios Human não se importam muito com a questão — estão ocupados demais construindo coisas.
>>
>> Sua vantagem evolutiva é simples: vivem menos, então fazem mais. Uma geração de Human muda mais o mundo que um século de qualquer outra raça.
>>
>> ## Aparência
>>
>> Variam enormemente — a maior diversidade física entre todas as raças, resultado de séculos de migração e adaptação.
>>
>> ## Papel na Era Atual
>>
>> Os Human lideram a maioria das expedições de descoberta. A [[Irmandade da Bruma]] é predominantemente humana. O Almirante **Serafin Davar** é humano.
>>

>>
>> ## Ancestralidade — Daggerheart
>>
>> ### Feature 1: Versatilidade
>> *Uma vez por sessão*, quando você fizer uma rolagem usando um Traço no qual você tenha uma Experiência relevante, você pode re-rolar um dos dados e ficar com o resultado mais alto.
>>
>> ### Feature 2: Ambição Sem Limites
>> Quando você ganha Esperança por superar um obstáculo que parecia impossível, ganhe **+1 Esperança adicional**.
