# Clank

> [!grid]
>> [!col]
>> > *"Perguntaram a ela se se sentia viva. Ela considerou por três dias. Depois disse: 'Não sei. Mas me importo com a resposta — isso não é suficiente?'"*
>>
>> ---
>>
>> ## Lore
>>
>> Os Clank são o maior segredo da civilização [[Dwarf]]. Por séculos, os melhores artífices das Raízes desenvolveram em silêncio uma técnica de unir um fragmento de [[Sistema de Magia|Filamento]] a um corpo construído — pedra, metal, madeira petrificada — criando algo que pensa, sente e age.
>>
>> Não são autômatos. Não são pessoas comuns. São algo para o qual [[Eryndal]] ainda não tem palavra adequada.
>>
>> Os [[Dwarf]] debatem intensamente a ética de sua criação. Alguns Clank trabalham livremente entre os clãs. Outros foram "desativados" — uma palavra que os Clank ativos consideram assassinato. A questão nunca foi resolvida.
>>
>> ## Aparência
>>
>> Variam enormemente dependendo do artífice que os construiu: corpos de pedra esculpida, metal moldado, ou madeira entalhada. Sempre com um núcleo visível onde o Filamento pulsa — uma luz suave que muda de intensidade com as emoções. Podem ser do tamanho de um humano pequeno ou de um Dwarf adulto.
>>
>> ## O Filamento Interno
>>
>> O Filamento que os anima é sempre o **Fio Cinza** — o de [[Morthus]]. Ninguém sabe se isso foi escolha dos artífices ou inevitabilidade. Os próprios Clank às vezes ouvem sussurros que não têm origem identificável. [[Morthus]] nunca comentou sobre sua relação com eles.
>>

>>
>> ## Ancestralidade — Daggerheart
>> *Equivalente: Clank*
>>
>> ### Feature 1: Corpo Construído
>> Você não precisa de ar, água ou comida para sobreviver — apenas de tempo de repouso para que seu Filamento interno se recarregue. Você é imune a venenos e doenças biológicas. Dano que reduziria você a 0 HP pode ser reparado com ferramentas adequadas em vez de cura mágica.
>>
>> ### Feature 2: Filamento Ancorado
>> Seu Filamento interno é estável — você não sofre os custos pessoais normais de usar magia do **Fio Cinza**. Além disso, *uma vez por sessão*, você pode "absorver" um efeito mágico direcionado a você, anulando-o e ganhando **1 ficha de Esperança**.
>>
