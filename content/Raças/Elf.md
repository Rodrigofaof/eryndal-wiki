# Elf

> [!grid]
>> [!col]
>> > *"Nós lembramos o que o mundo era. Isso não é sabedoria — é fardo."*
>>
>> ---
>>
>> ## Lore
>>
>> Os Elf foram os primeiros mortais criados em [[Eryndal]] — moldados pelas mãos combinadas de [[Solaren]] e [[Nyxara]], carregando tanto a clareza da ordem quanto a profundidade do mistério. Vivem centenas de anos e possuem memória perfeita.
>>
>> O problema com memória perfeita: você nunca esquece os erros. Os Elf mais velhos carregam o peso de guerras que os Human nem sabem que aconteceram.
>>
>> ## A Grande Biblioteca
>>
>> A obra mais importante dos Elf é a **Grande Biblioteca de Nyxara** em Auranthas — o maior repositório de conhecimento de [[Eryndal]]. Os [[Conselho dos Ancien]] a governam com mão de ferro.
>>
>> ## Postura Atual
>>
>> Os Elf estão divididos sobre a descoberta do [[Arquipélago das Brumas]]. Os mais velhos reconhecem algo naquelas ilhas — mas não dizem o quê. Os mais jovens querem explorar.
>>
>> ---
>>
>> ## Ancestralidade — Daggerheart
>>
>> ### Feature 1: Memória dos Filamentos
>> Você tem afinidade natural com todos os seis [[Sistema de Magia|Fios]]. Quando usar magia de qualquer Domínio, você pode adicionar +1 ao resultado **uma vez por longa descanso**.
>>
>> ### Feature 2: Longevidade e Perspectiva
>> *Uma vez por sessão*, quando o Narrador apresentar um perigo ou obstáculo, você pode perguntar: *"Já vi algo assim antes?"* — o Narrador deve dar uma informação verídica sobre a natureza do perigo.
>>
