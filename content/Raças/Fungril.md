# Fungril

> [!grid]
>> [!col]
>> > *"Não pensam individualmente. Cada Fungril é um pensamento de um ser maior que ainda não terminou de pensar."*
>> > — Ancião Syleth, notas pessoais
>>
>> ---
>>
>> ## Lore
>>
>> Nas profundezas mais baixas das Raízes — abaixo até de onde os [[Dwarf]] constroem seus túneis — existe uma rede de fungos que tem entre dois e três mil anos. Os Fungril são os corpos que essa rede desenvolveu para interagir com o mundo de superfície.
>>
>> Humanoides com pele em tons de bege, cinza ou violeta com manchas de bioluminescência, estruturas fungosas crescendo da cabeça como cabelos ou chapéus, e olhos completamente brancos que funcionam perfeitamente no escuro.
>>
>> ## Natureza Coletiva
>>
>> Cada Fungril individual é parcialmente conectado à rede de esporos da colônia de origem. Eles compartilham memórias difusas entre si — não pensamentos completos, mas impressões, sensações, estados emocionais. A morte de um Fungril não é sentida como tragédia pela colônia, mas como perda de um sentido.
>>
>> Isso os torna difíceis de entender para raças individualistas. E difíceis de manipular, porque enganar um é raramente enganar o grupo.
>>
>> ## Relação com os Dwarf
>>
>> Os [[Dwarf]] e os Fungril coexistem há milênios nas Raízes com uma relação que nenhum dos dois sabe nomear. Não são aliados formais. Não são inimigos. São simplesmente... vizinhos que respeitam os limites um do outro.
>>
>> O [[Clã de Pedra]] descobriu recentemente que a rede Fungril está *reagindo* ao que está acontecendo nas profundezas. A rede está contraindo, retirando-se para cima. Isso é, em termos de linguagem Fungril, equivalente a correr em pânico.
>>

>> ## Ancestralidade — Daggerheart
>> *Equivalente: Fungril*
>>
>> ### Feature 1: Rede de Esporos
>> Você pode se comunicar com outros Fungril ou com vegetação fúngica num raio de 100 metros sem falar. Em combate, se houver outro Fungril presente, vocês nunca são surpreendidos — a rede sempre avisa.
>>
>> ### Feature 2: Memória Coletiva
>> *Uma vez por sessão*, você pode "consultar a rede" — acessar memórias coletivas de Fungril que já estiveram em um lugar. Role **Instinto** (dificuldade 14): em sucesso, o Narrador revela informações históricas sobre o local que nenhum registro escrito conteria.
>>
